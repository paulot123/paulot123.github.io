BMO - Local AI Voice Assistant

June 2025 – August 2025

Overview

BMO is a fully local, privacy-focused AI assistant that combines offline speech recognition, text-to-speech, and on-device LLM inference. Designed as a mental health chatbot, it ensures complete privacy by processing everything locally without any cloud dependencies.

Media Gallery

Key Features

  • Integrated Mistral-7B Instruct using llama-cpp-python for efficient local inference without cloud dependencies
  • Developed FastAPI backend with modular REST endpoints, Pydantic models, and CORS support for frontend communication
  • Built React (Vite + React 19) frontend with client-side routing and Axios-based API integration
  • Implemented offline speech recognition and text-to-speech capabilities
  • Integrated Spotify API for music-related mental health features

Technologies Used

Python FastAPI React 19 Mistral-7B llama.cpp pygame Spotify API Vite

EpNext - TV Show Tracker

May 2025 – July 2025

Overview

EpNext is a full-stack TV episode tracker that helps users manage their TV show watching experience. With access to over 12,000 shows, users can search, subscribe, track watched episodes, and receive notifications for upcoming releases.

Media Gallery

Key Features

  • Built secure RESTful API using FastAPI to handle all application logic, including user authentication, data retrieval, and database operations
  • Utilized SQLAlchemy as the ORM for seamless interaction with a PostgreSQL database
  • Managed database schema changes and versioning with Alembic migrations
  • Implemented secure user registration and login system using JWT for stateless authentication
  • Integrated TVMaze API to fetch and cache show and episode data, including summaries, ratings, and schedules
  • Designed background scheduling system using APScheduler to automatically check for new episodes and update show information

Technologies Used

FastAPI React PostgreSQL SQLAlchemy JWT Alembic APScheduler TVMaze API
